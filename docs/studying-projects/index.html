Здесь представлены проекты, выполненные в ходе обучения на программе "Нейротехнологии и программирование".

1. Приложение для учёта расходов (Python)

Хакатон | Командный проект

Описание

Сайт для отслеживания личных финансов с функцией категоризации расходов и визуализацией статистики.

Технологии:

  • Python
  • SQLite

Функционал:

  • Добавление/редактирование расходов по категориям
  • Ежемесячная статистика с круговыми диаграммами

2. Мониторинг курса валют (Python)

Описание

Десктопное приложение для отслеживания курсов валют в реальном времени с уведомлениями о достижении пороговых значений.

Технологии:

  • Python
  • SQLite

3. Объектные модели предметных областей (Java)

Описание

Разработка объектно-ориентированных моделей для различных предметных областей с реализацией бизнес-логики.

Технологии:

  • Java 11+
  • ООП принципы (наследование, полиморфизм, инкапсуляция)
  • Паттерны проектирования

4. Классификация текстовых данных (Python)

Хакатон по NLP

Описание

Разработка многоклассового классификатора для автоматической категоризации текстовых документов.

Технологии:

  • Python
  • Библиотеки: scikit-learn, pandas, nltk/spaCy
  • Методы: TF-IDF, Word2Vec

Этапы:

  1. Предобработка текста (токенизация, лемматизация)
  2. Векторизация текстовых данных
  3. Обучение и оценка моделей
  4. Оптимизация гиперпараметров

5. RAG-система на базе GigaChat

Описание

Реализация Retrieval-Augmented Generation системы для интеллектуальных ответов на вопросы по специализированной документации.

Технологии:

  • Python
  • GigaChat API (или аналоги)
  • LangChain
  • Векторные базы данных (ChromaDB)
  • Embedding модели

Архитектура:

  1. Retriever: Поиск релевантных фрагментов документов
  2. Generator: Формирование ответов с использованием LLM
  3. Knowledge Base: Векторное хранилище документов

Применение:

  • Помощь для изучения научных статей
