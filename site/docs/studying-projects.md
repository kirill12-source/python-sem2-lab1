## 5. RAG-система на базе GigaChat

### Описание
Реализация Retrieval-Augmented Generation системы для интеллектуальных ответов на вопросы по специализированной документации.

### Технологии: <br>
- Python <br>
- GigaChat API (или аналоги) <br>
- LangChain <br>
- Векторные базы данных (ChromaDB) <br>
- Embedding модели <br>

### Архитектура: <br>
1. **Retriever**: Поиск релевантных фрагментов документов <br>
2. **Generator**: Формирование ответов с использованием LLM <br>
3. **Knowledge Base**: Векторное хранилище документов <br>

### Применение:
- Помощь для изучения научных статей
